A Movie Watchlist application that enables
users to:
1. Create their movie lists.
    a. Watched list.
    b. To-watch list.
    c. Rewatch list.
2. Follow others and see their movie lists.
3. Discover new films.
4. Make a personal favorite movie ordered list.
